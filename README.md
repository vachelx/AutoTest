#### Appium：
是一个开源测试自动化框架，可用于原生，混合和移动Web应用程序测试。

它使用WebDriver协议驱动iOS，Android和Windows应用程序；

UiAutomator测试框架是AndroidSDK自带的App UI自动化测试Java库；

appium在Android端基于WebDriver协议，利用Bootstrap.jar，最后通过调用UiAutomator的命令，实现App的自动化测试。
